``specview``: Spectrum viewer and model fitting
===============================================

``specview`` is a package that will form the basis for spectal viewing
and model fitting
in order to aid data
analysis for the data supported by `STScI <http://www.stsci.edu/>`_ (e.g.,
`HST <http://www.stsci.edu/hst/>`_ or `JWST <http://www.stsci.edu/jwst/>`_).
